# Dino-Run-

**Assalamualaikum.**


The following is a HTML/CSS/JavaScript project which is the famous Dinosaur game which we can play in the browser (Chromium based) when the Internet goes out. I have added a few extra touches like:

 1. A custom start screen and a game over screen
 2. A high score counter
 3. Custom background music in loop

==============================================

The languages and tools I have used are as follows:

 1. HTML/CSS/JavaScript
 2. VS Code Text Editor
 3. Live Server (To auto-reload changes)

==============================================

To run the game locally, download the whole repo, move everything into a folder and then, run it using the Live Server VS Code extension. This is because due to the usage of background music; web browsers have a strict AutoPlay policy which won't allow the game to work by just opening the index.html file. 
Or you can just open the following link at [https://phantoma03.github.io/Dino-Run-/](https://phantoma03.github.io/Dino-Run-/)
